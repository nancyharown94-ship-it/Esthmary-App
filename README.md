# Esthmary Platform (استثماري) 🤝📱

An innovative mobile investment matchmaking platform inspired by the Shark Tank concept, designed to seamlessly bridge the gap between small business owners/startups and potential investors.

## 🚀 Key Features
* **Entrepreneur-Investor Matchmaking:** A streamlined workflow allowing startups to pitch ideas and investors to find lucrative opportunities.
* **Proposal & Project Management:** Full CRUD operations for creating, editing, and deleting business projects and investment proposals.
* **Offer System:** Built-in features to accept or decline investment offers with dynamic updates.
* **Secure Backend Validation:** Utilizes robust database rules and cloud architecture to ensure user data integrity and privacy.

## 🛠️ Tech Stack
* **Frontend & Mobile App:** Flutter, Dart
* **Backend & Database:** Firebase (Firestore, Authentication, Cloud Storage)
* **Architecture:** Professional state management and structured UI components.

## 📁 How to Run Locally
1. Ensure you have the Flutter SDK installed on your machine.
2. Clone this repository.
3. Navigate into the project folder: `cd flutter_application_1`
4. Run `flutter pub get` to fetch all necessary package dependencies.
5. Connect an Android/iOS emulator or a physical device.
6. Run `flutter run` to launch the application.
